# Pickle for Java
Maybe, a much-needed extension to
[Cucumber-JVM](https://cucumber.io/docs/installation/java/) and
[Gherkin](https://cucumber.io/docs/gherkin/).

## Build

```bash
mvn install --define skipTests
```

## Test

```bash
mvn test
```
